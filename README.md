## 💡 1. Blender Add-on Tool【画像エフェクト生成ツール】

### ▶ 概要
非デザイナーでも、画像を選択→ボタン一つでエフェクトを生成できるアドオンを制作。  
スパイラル状や円形に画像を配置し、AfterEffects風のループアニメーションを出力できます。

- 自己学習 → 実装 → UI改善まで一貫して担当
- 技術：Python（Blender API）、GUI設計
